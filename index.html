Personal Security Checkup

A hands-on tool to check your password strength, breach exposure, and public digital footprint.

Password Strength Live scoring based on length, character variety, and common-password detection.
Breach Check See if your password has appeared in known data breaches, using a privacy-preserving lookup technique (k-anonymity) — your password is never sent over the internet.
Digital Footprint Check if a username exists on GitHub automatically, plus quick links to manually verify Instagram, Twitter/X, Reddit, and LinkedIn. Note: usernames are unique per platform — a result reflects exactly one account, never multiple people sharing a handle.
